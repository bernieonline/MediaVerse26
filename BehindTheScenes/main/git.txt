Which branch am I in - git branch
Switch to a different branch - git checkout your_desired_branch
to see all branches - git branch -a
To see your changes since yesterday - git log --since="yesterday" - make sure you are in the desired branch when you run it
to see the actual changes  - git diff master@{yesterday}..master
