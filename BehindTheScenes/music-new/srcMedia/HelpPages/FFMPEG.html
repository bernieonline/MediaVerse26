Welcome to the MediaManager FFMPEG Actions Section

This is where we can test and possibly repair video files as well as converting to other formats

Here's how it works:

  • We should begin with the Master File
  • Select a file from the master location on the server - When testing for errors
  • A txt file will be created on the server in the same folder as the master file listing the main errors as well as media data from MediaInfo
  • Not all errors are noticeable on playback so it 's best to try and play it first before you try for a fix
  • If you do that you will know where the error is and how severe
  • The Test a Folder does the same as the above but in batches
  • Repairing files can be difficult as audio and video may be affected, the usual approach is to recreate the file as an mp4 with few adjustments
  • When repairing files its best to copy the file to be repaired to a temporary location and work on the copy
  • The option to view a file details shows any reported errors as well as metadata
  • The Video Proc action is Third Party Software featuring AI enhancement but can be vert slow to run

Lets Get Started

Its best to start slowly and become familiar with the process Error Files will be created and you may want to save them .
