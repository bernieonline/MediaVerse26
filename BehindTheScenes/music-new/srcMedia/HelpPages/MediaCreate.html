Welcome to the MediaManager - Create Media records

This together with the Creation of Master Records is the heart of the system.

The purpose of the software is to search for Media files and record their location

A search will be initiated from a folder and the sets of media files discovered will be grouped under a collection name referencing that location with device and datetime details.

Here's how it works:

  • First ensure that you are searching for groups of files NOT in a Master set
  • Creating Master Sets (A similar process), would normally be done first, aiming perhaps at three copies in three different locations
  • One of the main purposes is to identify surplus copies of media files and then take action to either remove, re-organze, synchronize or delete freeing up space and avoiding unneccesary duplication
  • To create a collection or multiple collections choose Select Location
  • Collections are divided by Media Type. If a search discovers video and Music files two collections will be created, or more for different types
  • A File Explorer window will open and the user will navigate to a folder where the search will begin.
  • On choosing the folder, the program will run in the background drilling deeply into your file system.
  • The LCD Displays at the top of the page show system resource levels, the start time of the run and in the bottom right corner is a progress bar
  • Large data sets can take more than an hour to run but the display will confirm activity
  • Once it completes you can examine it in the Reports section
